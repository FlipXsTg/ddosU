------------------------------
	Clone repository
------------------------------
1.Install git:
	$sudo apt-get install git
2.Clone files: 
	$git clone https://github.com/unaiiM/nodeDOS.git
------------------------------
	Installation
------------------------------
1.Install requeriments:
	$sudo bash requeriments.sh
2.Start ddos! :
 	$node dos.js


----------------------------------------
	Installation for Android
----------------------------------------
1.Open termux and install git:
	$pkg install git
2.Clone files:
	$git clone https://github.com/unaiiM/nodeDOS.git
3.Install requeriments:
	$bash requerimentsAndroid.sh
4.Start dos:
	$node dos.js
